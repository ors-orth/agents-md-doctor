# AGENTS.md Doctor

AGENTS.md Doctor is an early-stage project for checking, maintaining, and improving `AGENTS.md` files.

## Status

This repository is currently being prepared for open source development. The implementation and public API are not stable yet.

## Goals

- Help teams keep `AGENTS.md` instructions clear, consistent, and actionable.
- Detect common problems in agent instruction files.
- Provide practical feedback that can be used in code review and documentation workflows.

## Getting Started

There is no installable package yet. Development instructions will be added as the implementation takes shape.
