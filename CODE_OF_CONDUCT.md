# Code of Conduct

This project is intended to be a respectful, practical, and welcoming place to collaborate.

Participants are expected to:

- Be considerate and constructive.
- Assume good intent while still addressing concrete problems directly.
- Avoid harassment, personal attacks, discriminatory language, or sustained disruption.
- Respect maintainers' time and project scope.

Maintainers may remove comments, issues, pull requests, or contributions that do not meet these expectations. Repeated or severe violations may result in being blocked from participating in the project.
